# Sankalp Jadhav — Portfolio

Personal portfolio site for Sankalp Jadhav, a full-stack developer focused on AI-integrated web experiences.

**Live site:** [sankalp-jadhav.vercel.app](https://sankalp-jadhav.vercel.app/)

## About

A single-page portfolio built around a film-timeline visual concept — sections are numbered like reel frames (01, 02, 03...) instead of a standard nav. Covers background, experience, editorial leadership, selected projects, and contact.

## Sections

- **Hello, Internet** — intro and headline
- **A Quick Read** — background and academic info
- **Experience** — SyncEdge web development internship
- **Campus Leadership** — Editor-in-Chief, *That's IT* annual tabloid
- **Selected Work** — AI Content Calendar, Menstrometer, ISP Billing
- **The Toolbox** — tech stack
- **Your Turn** — contact / socials

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deployment)

## Featured Project

**[AI Content Calendar](https://github.com/SankalpJadhav28/Ai-Content-Calendar)** — an AI-powered content planning workspace built with Next.js, Supabase, and the Groq API (Llama 3.3 70B).
[Live demo →](https://ai-content-calendar-seven.vercel.app/)

## Running Locally

```bash
git clone https://github.com/SankalpJadhav28/<repo-name>.git
cd <repo-name>
npm install
npm run dev
```

Visit `http://localhost:3000`.
